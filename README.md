# EazyMart
EazyMart is an ecommerce project built using React. It provides users with a platform to browse and purchase products online.

## Features

- Browse products by category
- Search for products by name or category
- Add products to cart and proceed to checkout
- User authentication and registration
- Responsive design for mobile and desktop

## Installation

To run EazyMart locally, follow these steps:

1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/your-username/EazyMart.git

## Change into the project directory:
cd EazyMart

## Install dependencies using npm
npm install

## Start the development server:
npm start
